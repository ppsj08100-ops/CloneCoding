# 롯데월드 민속박물관 클론코딩

롯데월드 민속박물관 웹사이트를 참고하여 제작한 HTML/CSS/JavaScript 클론코딩 작업물입니다.  
메인 슬라이드, 전시 안내, 이용 정보, 공지 영역 등 메인 페이지 중심으로 구성했습니다.

## 작업 내용

- HTML 시맨틱 구조 작성
- CSS를 활용한 메인 페이지 레이아웃 구현
- Swiper 라이브러리를 활용한 비주얼 슬라이드 구현
- JavaScript를 활용한 슬라이드 재생/정지 기능 구현
- 마우스 오버에 따른 콘텐츠 배경 이미지 변경
- 이미지 리소스를 활용한 섹션별 화면 구성

## 사용 기술

- HTML5
- CSS3
- JavaScript
- Swiper
- XEIcon

## 폴더 구조

```text
lotteworld/
├── css/
│   ├── reset.css
│   └── main.css
├── images/
├── js/
│   └── main.js
├── index.html
└── README.md
```

## 실행 방법

`index.html` 파일을 브라우저에서 열어 확인할 수 있습니다.

```text
lotteworld/index.html
```

## 작업 목적

실제 사이트의 메인 페이지 구성을 분석하고, HTML/CSS 레이아웃과 JavaScript 인터랙션을 함께 연습하기 위해 제작했습니다.

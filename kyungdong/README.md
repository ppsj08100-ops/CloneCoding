# 경동 클론코딩

경동 웹사이트를 참고하여 제작한 HTML/CSS 클론코딩 작업물입니다.  
메인 비주얼, 사업 영역, 기술 혁신, 홍보 센터, 푸터 영역으로 구성되어 있습니다.

## 작업 내용

- HTML 시맨틱 구조 작성
- CSS를 활용한 레이아웃 및 스타일 구현
- Swiper 라이브러리를 활용한 메인 비주얼 슬라이드 구성
- XEIcon 아이콘 CDN 사용
- 이미지 리소스를 활용한 섹션별 화면 구성

## 사용 기술

- HTML5
- CSS3
- JavaScript
- Swiper
- XEIcon

## 폴더 구조

```text
kyungdong/
├── css/
│   ├── reset.css
│   └── main.css
├── images/
├── index.html
└── README.md
```

## 실행 방법

`index.html` 파일을 브라우저에서 열어 확인할 수 있습니다.

```text
kyungdong/index.html
```

## 작업 목적

기존 웹사이트의 레이아웃과 비주얼 구성을 분석하고, HTML/CSS 구조화 및 메인 페이지 구현 연습을 목적으로 제작했습니다.
